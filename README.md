# Business Insights 360 – Power BI Dashboard

## Overview
This project presents a Business Intelligence dashboard built using Power BI to analyze sales performance, profitability, and forecast accuracy across different regions, segments, and customers.

## Key Insights
- Net Sales: $267.98M
- Gross Margin: 37.1%
- Net Profit: -0.85%
- Forecast Accuracy: 72.99%

## Dashboard Views
1. Finance View – Profit & Loss analysis
2. Sales View – Customer and product performance
3. Marketing View – Segment growth and profitability
4. Supply Chain View – Forecast accuracy and inventory risk
5. Executive View – High-level KPI overview

## Tools Used
- Power BI
- Power Query
- DAX
- Data Modeling

## Features
- Interactive filters for region, market, segment, and customer
- Profitability and growth matrix
- Forecast accuracy analysis
- Net Sales trend analysis
